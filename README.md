# Next.js Blog Project

A modern blog application built with Next.js, TypeScript and React.

## Features

- Blog listing and detail pages
- Admin panel for creating posts
- Edit and delete functionality
- Responsive design
- RESTful API routes

## Tech Stack

- Next.js 14
- React
- TypeScript
- CSS3

## Getting Started

```bash
git clone https://github.com/username/nextjs-blog-project.git
cd nextjs-blog-project
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
